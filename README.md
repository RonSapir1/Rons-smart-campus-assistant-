# 🎓 Smart Campus Assistant

מערכת בינה מלאכותית חכמה לקמפוס, המבוססת על ארכיטקטורת RAG (Retrieval-Augmented Generation) להבטחת תשובות אמינות, מדויקות וחסינות-הזיות (Zero Hallucinations). המערכת ארוזה כולה בקונטיינרים (Dockerized) ומוכנה לפריסה מהירה בענן (Cloud-Native).



## 🏗️ תיאור הארכיטקטורה
המערכת בנויה בארכיטקטורת Microservices קלת-משקל:

1. **Frontend (ממשק משתמש):** שרת אינטרנט מהיר (Nginx) המגיש ווידג'ט צ'אט מרחף, ריאקטיבי ומעוצב (Premium UI) הבנוי ב-Vanilla HTML/CSS/JS ללא תלויות חיצוניות.
2. **Backend (שרת ולוגיקה):** שרת מבוסס Python ו-**FastAPI**. השרת מנהל את התקשורת מול ה-AI, מתשאל את מסד הנתונים, ומפעיל מנגנון Fallback אוטומטי במקרה של חוסר ודאות מצד המודל (מניעת הזיות).
3. **Data & AI (נתונים ומודל):** * **מסד נתונים (SQLite):** מאחסן את נתוני האמת של הקמפוס. הנתונים נשמרים בצורה בטוחה באמצעות Docker Volumes.
   * **מנוע AI:** אינטגרציה מול **Gemini 2.5 Flash** בתצורת JSON Mode נוקשה המחייבת החזרת תשובה, קטגוריה וציון ביטחון.

## ⚙️ משתני סביבה נדרשים (Environment Variables)
יש ליצור קובץ בשם `.env` בתוך תיקיית ה-`backend/`, ולהגדיר בתוכו את מפתח ה-API שלכם עבור שירותי Google Gemini:

```env
GEMINI_API_KEY=your_api_key_here

🚀 הוראות התקנה והרצה (פקודה אחת)
בזכות השימוש ב-Docker, אין צורך בהתקנת סביבות פייתון מקומיות או הורדת ספריות.
דרישות קדם: ודאו ש-Docker ו-Docker Compose מותקנים ופועלים על המחשב שלכם.

הפעלת המערכת:
פתחו את הטרמינל בתיקיית השורש של הפרויקט (היכן שנמצא קובץ ה-docker-compose.yml) והריצו את הפקודה הבאה:
docker-compose up --build
